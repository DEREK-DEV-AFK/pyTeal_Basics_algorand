# Basics of pyTEAL 

through pyTEAL developers can write Algorand Smart Contracts (ASC) in python

## Dependencies 
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
# pyTEAL 
pip install pyteal
```
